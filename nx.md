# NX

- Build system + monorepo toolkit for JS/TS ecosystem
- It focuses on:
  - Structure
  - Speed
  - Consistency

## Core concepts

### 1. Monorepo

- A single repo containing multiple apps + libraries
  - Shared code
  - Shared tooling
  - Unified CI/CD

```
apps/
    web/
    admin/
    api/
libs/
    ui/
    utils/
    auth/
```
